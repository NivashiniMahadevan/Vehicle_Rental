# 🚗 Vehicle Rental Management System

## 📌 Overview

The **Vehicle Rental Management System** is a web-based application designed to help administrators manage and monitor a vehicle rental business efficiently. It provides centralized control over customers, vehicles, bookings, rentals, payments, maintenance, and fleet lifecycle.

This system is built as an **admin dashboard**, focusing on business operations rather than customer-facing booking.

---

## 🎯 Features

### 🔹 Dashboard

* Key metrics: Total Vehicles, Customers, Bookings, Active Rentals
* Booking trends and revenue insights
* Business intelligence widgets (idle vehicles, pending payments, etc.)

### 🔹 Customers Management

* Add, view, and manage customer details
* Stores license, contact, and address information

### 🔹 Vehicles Management

* Add and track vehicles
* Status tracking (Available, Rented, Maintenance)

### 🔹 Bookings

* Create bookings with date validation
* View all booking records
* Status tracking (Confirmed, Completed, Cancelled)

### 🔹 Rentals

* Manage active rentals
* Track return status

### 🔹 Payments

* Record payments
* Track payment status (Paid, Pending)

### 🔹 Maintenance

* Log vehicle issues
* Track service status

### 🔹 Fleet Lifecycle

* Monitor vehicle stages (Active, Maintenance, Retired)
